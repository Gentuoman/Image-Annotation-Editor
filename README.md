# 使用說明

這是用來在圖片上加可拖曳的說明框。常見的使用情境是為照片標示細節、標出重點或做說明。
完成後可以輸出成幾種格式：直接貼進網站的 HTML 代碼、給海報用的高解析向量檔（SVG）、貼 LINE / Facebook 用的圖檔（PNG / JPG），以及之後還想修改用的設定檔（JSON）。

網址 https://gentuoman.github.io/Image-Annotation-Editor/

---

## 30 秒上手

打開網頁後依序做四件事：

1. 在右側面板上方選「上傳檔案」，把要加說明的圖片拖進虛線框（或點擊選擇）。
2. 點上方工具列的「＋ 新增框」，畫面上會出現一個白色說明框。
3. 點那個框讓它被選中（外圍出現藍框和八個方塊），右側面板下方就會展開所有可調設定，依需要修改文字、顏色、大小。
4. 點上方「輸出 ▾」，選一個格式輸出。

就這樣。下面是細節。

## 介面說明

畫面分三個區塊：頂端工具列、左邊大畫布、右邊側邊欄。

**頂端工具列**：左邊是新增框 / 複製 / 刪除三個按鈕，右邊是「匯入 JSON」和「輸出 ▾」下拉選單。

**左邊大畫布**：圖片放在這裡，所有說明框都加在這上面。還沒載入圖片時會顯示一個提示框。

**右邊側邊欄**：上面是底圖來源切換（上傳檔案 / 貼網址），中間是已加入的說明框列表，下面是選中說明框的詳細設定。沒選任何框時，下面那塊會收起來。

---

## 載入圖片

兩種方式擇一：

**上傳本地檔案**——選「上傳檔案」，可以點虛線框選檔案，或直接把圖檔從桌面拖進去。圖片會以 base64 編碼方式內嵌進工具裡，輸出時也會內嵌進輸出檔，意思是輸出檔可以離線使用，不需要連網就能顯示，但檔案會比較大。

**貼網址**——選「貼網址」，把線上圖片的 URL 貼進去再按「載入」。HTML / SVG 輸出會直接參照那個網址（檔案小，但讀者需要連網才能看到）。需注意：用網址載入的圖片若要輸出 PNG / JPG，圖片來源伺服器必須允許跨域存取，不是每個網站都允許。如果輸出失敗會跳訊息。

不知道選哪個就用「上傳檔案」，最不會出問題。

---

## 加入與編輯說明框

點工具列「＋ 新增框」，畫面中央會出現一個有黑色邊框的白色框，預設文字是「說明文字」。新增後框會自動被選取，右側面板下方展開所有設定。

**標題列**：勾選「顯示標題列」會在內文上方多一個粗體的標題欄位。沒勾就只有單欄內文。標題與內文是兩個不同的輸入欄位，可以獨立修改。

**內文**：直接在「內文」textarea 輸入文字。要換行就按 Enter，輸出時會自動處理。

**外觀**：背景色（含 0–100% 的透明度拉桿）、文字色、字級。背景設透明度可以讓底下的圖片若隱若現，適合做柔和的標註。

**框線**：顏色、粗細（0 表示完全不畫邊框）、圓角（0–40px，0 是直角，數字越大越圓）。

---

## 移動與縮放

選中框之後（外圍藍框和八個小方塊出現了就是選中狀態）：

要移動，直接用滑鼠抓住框體任何地方拖動即可。

要縮放，抓住四個角或四個邊上的藍色小方塊拖動——角落是雙向縮放，邊上是單向縮放（只動寬或只動高）。

要精確定位，右側面板最下方有「位置與尺寸」四個輸入框（X / Y / 寬度 / 高度，單位都是百分比，0–100），手動填數字也可以。

---

## 引線指向

如果想讓說明框明確指向圖片中的某個位置（例如「就是這個茶壺」、「請看落款」），就用引線：

勾選右側面板的「啟用引線指向」，會多出一條從框邊延伸出去的線，盡頭有一個紅色圓點。把那個紅色圓點拖到你要指的位置即可，引線的另一端會自動連到框的最近邊緣，不需要自己對準。

引線可調的有：線色、線寬、是否顯示箭頭（預設有箭頭）。

引線是逐框獨立的——可以有些框有引線、有些沒有，混用沒問題。

紅色端點只有在「該框被選中、且引線啟用」時才會顯示。看不到端點時點一下框讓它被選中即可。

---

## 同時管理多個框

側邊欄中段的「說明框列表」會列出所有已加入的框，點任一筆就跳到那個框。每筆會顯示底色色塊和文字預覽，方便辨認。

要做一系列風格一致的標籤，先把第一個框調好，選中後按「複製」（或 ⌘ / Ctrl + D），會在原框旁邊出現一個一模一樣的，再改文字和位置就好。

刪除：選中框後按「刪除」按鈕，或直接按鍵盤 Delete / Backspace。

---

## 鍵盤快捷鍵

`Delete` / `Backspace` 刪除選中的框  ·  `Esc` 取消選取  ·  `⌘ + D`（Mac）或 `Ctrl + D`（Windows）複製選中的框

---

## 自動存檔與長期保存

每次修改都會自動存進瀏覽器，重整頁面或關掉再打開都不會掉資料。但這個自動存檔是綁在「同一個瀏覽器、同一台電腦」的——換電腦或換瀏覽器就看不到。

需要跨裝置或長期保存，請用「輸出 → 儲存設定檔（.json）」匯出 JSON 檔案；之後「匯入 JSON」就能還原所有設定繼續編輯。重要的工作建議都存一份 JSON 備份。

---

## 輸出格式怎麼選

工具列「輸出 ▾」展開後有五個選項，挑使用情境決定：

**HTML 代碼**——貼進故居網站、CMS、部落格時用這個。點下去會跳出一個預覽視窗，裡面是一段 HTML 代碼，按「複製到剪貼簿」就能貼進 CMS 的編輯器；按「下載為 .html」會把代碼存成檔案。代碼經過設計只用最基本的 HTML 標籤（div、p、b、i、img、br 等），全部 inline style，能相容絕大多數 CMS 的標籤白名單。

**SVG**——向量格式，可以放大到任意尺寸不會模糊。用來做海報、印刷品、需要高解析度的場合。

**PNG**——一般用途的圖檔，會保留背景透明度設定。適合貼進 LINE、Facebook、簡報、Word 文件。

**JPG**——比 PNG 小很多，但不支援透明（會壓上白底）。適合需要省空間又是照片性質的圖。

**儲存設定檔（.json）**——不是圖片，是一份「目前所有設定」的存檔。日後想再修改就匯入這個檔，所有狀態（圖片、所有框的位置、文字、顏色、引線等等）都會回來。

## 常見問題

**圖片網址貼了但載入失敗**
通常是網址錯誤、圖片不存在、或那個網站不允許跨網域存取。先確認網址能在瀏覽器直接打開；如果可以但這個工具讀不到，就把圖片下載下來，改用「上傳檔案」方式載入。

**輸出 PNG 或 JPG 時跳「輸出失敗」訊息**
如果是用「貼網址」載入的圖片，輸出點陣圖需要對方伺服器允許跨域存取，多數圖庫站不允許。解法：把圖下載下來，改用「上傳檔案」載入再輸出。上傳的本地圖片永遠可以正常輸出。

**重整頁面後資料不見了**
若使用了非常大的圖（base64 超過 5MB），可能超過瀏覽器自動存檔的容量上限。瀏覽器是靜默拒絕，沒有訊息提醒。預防方法：圖片先壓到 2–3 MB 以內，或重要工作隨手「儲存設定檔（.json）」備份。

**框被拖出畫面外**
正常情況下拖曳和縮放都有限制，不會超過圖片邊界。萬一真的卡住，用右側「位置與尺寸」欄位直接輸入數字（X 和 Y 在 0–100 之間），就能把框拉回來。

**字看起來太小或太擠**
字級太小：在「字級」欄位調大（預設 14，普通網頁正文約 14–16）。框太擠：拖角放大框；或縮短內文；或挑圖片中相對單純的區域擺放。

**多個瀏覽器分頁同時開會怎樣**
不會壞，但會互相覆蓋——最後存檔的那個分頁會留下來，其他分頁的修改會被吃掉。建議一次只開一個分頁編輯。

**HTML 代碼貼進 CMS 之後排版跑掉**
代碼本身是乾淨的 inline style 結構，跑掉通常是 CMS 的 wysiwyg 編輯器在「視覺模式」下重新格式化所致。解法：找到 CMS 編輯器的「原始碼模式」或「HTML 模式」（通常是個 `<>` 圖示），切過去再貼，就不會被改寫。
